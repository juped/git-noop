#+TITLE: git-noop

* Overview

~git-noop~ is an extension to ~git~ which performs no operation.

* Usage

#+BEGIN_EXAMPLE
% git noop
#+END_EXAMPLE

* Installation

Place the ~git-noop~ executable somewhere on your PATH.

* Examples

#+BEGIN_EXAMPLE
% git noop
#+END_EXAMPLE
